# Korean Installment Savings Calculator

Status: Implemented

## Overview

### Purpose

정기적으로 같은 금액을 납입하는 적금 시나리오의 납입 원금, 세전 이자, 예상 세금과 만기 수령액을 계산한다. 월·연 납입, 단리·월복리, 기간 초·말 납입과 세금 조건의 차이를 한 화면에서 설명한다.

## User Problem

적금 가입 전에는 표시 금리만으로 실제 이자와 세후 만기액을 판단하기 어렵다. 납입금마다 이자가 붙는 기간이 다르고 납입 시점, 이자 방식, 과세 조건에 따라 결과가 달라지므로 동일한 가정 아래에서 투명하게 비교할 도구가 필요하다.

## Requirements

- 고정 납입액을 월 또는 연 단위로 납입한다.
- 기간은 개월 또는 년으로 입력해 내부에서 개월 수로 정규화한다.
- 단리와 월복리를 지원하고 납입 시점을 기간 초 또는 기간 말로 구분한다.
- 비과세, 일반과세, 사용자 지정 간이 세율을 지원한다.
- 납입 원금, 세전 이자, 예상 세금, 세후 이자, 세전·세후 만기액과 월별 내역을 같은 계산 결과에서 제공한다.
- 특정 금융상품의 실제 만기액을 보장하거나 우대금리, 중도해지, 수수료를 모델링하지 않는다.

## Formula

`C`는 정기 납입액, `r`은 연 이자율의 소수값, `N`은 전체 개월 수, `t`는 각 납입금이 만기까지 이자를 얻는 개월 수다.

단리는 납입금별로 `C × r × t / 12`를 계산해 합산한다. 이미 발생한 이자에는 다시 이자를 붙이지 않는다.

월복리는 납입금별로 `C × (1 + r / 12)^t`를 계산해 합산한다. 월별 발생 이자가 다음 달 원금과 함께 이자를 얻는다. 0%에서는 두 방식 모두 세전 만기액이 납입 원금과 같다.

기간 초 납입은 해당 납입 주기의 첫 시점에 넣어 한 주기 더 이자를 얻고, 기간 말 납입은 주기가 끝난 뒤 넣는다. 월 납입은 매월, 연 납입은 12개월마다 실행한다.

일반과세 기본값은 15.4%다. 국세청이 안내하는 일반 이자소득 원천징수 14%와 국가법령정보센터의 지방소득세 특별징수 산식(소득세 원천징수액의 10%)을 합친 단순 추정이다. 실제 과세 여부와 세액은 가입자, 상품과 법령 적용에 따라 달라질 수 있다.

계산은 Decimal.js 고정밀 값을 유지하고 표시할 때만 원 단위 반올림한다. `세전 만기액 = 납입 원금 + 세전 이자`, `세후 만기액 = 납입 원금 + 세전 이자 - 예상 세금`이 항상 일치해야 한다.

## Validation

- 숫자 입력은 빈 값, 부호, 지수 표기, `Infinity`, `NaN`, 잘못된 쉼표와 허용 범위 밖 값을 거부한다.
- 연 납입은 12개월 이상의 완전한 연 단위 기간만 허용해 부분 연도의 납입 횟수를 모호하게 만들지 않는다.
- 사용자 지정 세율은 해당 옵션을 선택했을 때만 필수다.
- 제출 시 모든 필드를 검사하고 첫 오류 입력으로 포커스를 옮긴다. 입력값은 보존하고 오류를 연결된 설명으로 제공한다.

## Edge Cases

- 0% 금리는 나눗셈이나 `NaN` 없이 이자와 세금이 0원이다.
- 기간 초 결과는 같은 양수 조건의 기간 말 결과보다 크다.
- 비과세는 세금을 0원으로 만들고 세전·세후 만기액을 같게 한다.
- 1,200개월 계산도 유한한 결과와 1,200개 이하의 표 행을 만든다.

## Mobile UX

입력, 핵심 세후 만기액, 보조 결과, 월별 내역, 설명 순서로 한 열에 배치한다. 모든 컨트롤은 영구 라벨과 44px 이상의 조작 영역을 사용한다. 표는 관계를 유지한 채 수평 스크롤한다.

## Desktop UX

입력과 결과를 나란히 배치하고 결과를 화면 상단에 유지한다. 설명은 읽기 쉬운 폭으로 제한하며 비교 콘텐츠는 카드 그리드로 확장한다.

## Accessibility

키보드로 모든 입력과 `details`를 조작할 수 있다. 포커스가 보이고, 도움말과 오류를 `aria-describedby`로 연결한다. 오류 요약은 `role=alert`, 계산 완료는 `aria-live=polite`로 알린다. 표는 caption, 열 머리글과 행 머리글을 제공하며 색만으로 의미를 전달하지 않는다.

## SEO

경로는 `/finance/savings`다. 고유 한국어 제목·설명, canonical, Open Graph, Twitter/X 메타데이터를 제공한다. 보이는 breadcrumb와 일치하는 `BreadcrumbList`, 페이지 내용과 일치하는 `WebPage`만 구조화 데이터로 제공한다.
