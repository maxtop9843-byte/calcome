# Korean Loan Calculator Foundation

Status: Implemented

## Overview

대출 금액, 고정 연이율, 기간과 상환 방식을 입력해 원리금균등, 원금균등, 만기일시상환의 월 납부액과 전체 비용을 비교하는 계산기다. 이후 금융 계산기가 따를 서버 페이지, 독립 계산 모듈, 검증, 접근 가능한 입력·결과, 설명 콘텐츠, SEO와 테스트 구조의 기준을 세운다.

## Requirements

- 기간은 개월 또는 년으로 입력하고 내부에서 개월 수로 정규화한다.
- 원리금균등과 원금균등은 월별 상환 일정을 제공한다.
- 총 상환액, 총 이자, 대표 월 납부액과 상환 방식별 요약을 제공한다.
- 만기일시상환은 만기 원금 부담을 명확히 설명한다.
- 계산 로직을 React와 Next.js에서 분리하고 Decimal.js로 중간 반올림 없이 계산한다.

## Inputs

| 입력      | 단위         | 기본값        | 범위            |
| --------- | ------------ | ------------- | --------------- |
| 대출 금액 | 원           | 100,000,000원 | 10,000원~1조원  |
| 연 이자율 | %            | 4.5%          | 0~100%          |
| 대출 기간 | 개월 또는 년 | 30년          | 1~1,200개월     |
| 상환 방식 | 선택         | 원리금균등    | 세 방식 중 하나 |

## Outputs

- 원리금균등: 일정한 월 납부액과 전체 상환 일정
- 원금균등: 첫 달과 마지막 달 납부액, 감소하는 전체 상환 일정
- 만기일시상환: 매월 이자, 총 비용과 만기 원금 상환 설명
- 모든 방식: 총 상환액과 총 이자

## Formula and assumptions

월 이자율은 명목 연이율을 12로 나눈다. 원리금균등은 표준 연금 현재가치 공식을 사용하고, 원금균등은 원금을 전체 개월 수로 나눈다. 만기일시상환은 만기 전까지 원금이 유지된다. 매월 말 납부, 고정 금리, 동일한 월 길이를 가정하며 수수료, 세금, 거치, 중도상환과 연체는 제외한다. 0% 금리는 별도 분기로 처리한다.

## Accessibility and mobile UX

영구 라벨, 단위, 도움말, 연결된 오류, 첫 오류 포커스, 계산 완료 라이브 알림을 제공한다. 결과 표는 caption과 행·열 머리글을 사용하고 작은 화면에서는 수평 스크롤한다. 핵심 입력과 결과는 모바일에서 세로로, 넓은 화면에서 나란히 배치한다.

## SEO and internal links

경로는 `/finance/loan`이며 고유 canonical, Open Graph와 Twitter 메타데이터를 제공한다. 보이는 내용과 일치하는 WebPage 및 BreadcrumbList 데이터만 사용한다. 복리 계산기로 연결하고 적금·예금 계산기는 관련 계산기 섹션 안에서만 Coming Soon으로 표시한다.
