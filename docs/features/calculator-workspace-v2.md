# Calculator Workspace V2

## 목적

CalCome 계산기를 빠르게 입력하고 핵심 결과부터 확인하는 작업 공간으로 개선한다. 원칙은 “입력은 단순하게, 결과는 핵심부터, 상세 정보는 필요할 때”이다.

## 작업 공간과 결과 계층

- 데스크톱과 여유가 있는 태블릿에서는 설정과 결과를 36:64 비율의 두 열로 배치한다.
- 설정 패널은 데스크톱에서만 계산기 영역 안에 고정되며 헤더나 푸터와 겹치지 않는다.
- 계산과 초기화 동작을 설정 패널 하단에 함께 제공한다.
- 결과 영역은 세 개의 핵심 지표를 먼저 보여 주고 차트, 상세 표, 부가 가정을 뒤에 둔다.
- 모바일에서는 설정, 계산, 핵심 결과, 차트, 상세 내역, 교육 콘텐츠 순서의 한 열로 흐른다.

## 홈페이지

검색을 핵심 진입점으로 유지한다. 인기 계산기는 복리, 대출, 예금, 적금, CAGR 순서이며 데스크톱과 태블릿에서 두 열, 모바일에서 한 열로 표시한다. 카드 높이와 여백을 줄여 첫 화면에서 더 많은 선택지를 비교할 수 있게 한다.

## 차트 원칙

복리 계산 결과가 있을 때만 Recharts 기반 시계열 선 차트로 누적 납입 원금과 예상 총자산을 비교한다. 기간이 길면 연도 축 레이블을 자동으로 줄이고, 금액 축은 읽기 쉬운 축약 원화로 표시한다. 포인터와 터치 툴팁은 연도, 원금, 예상 자산, 누적 이자를 제공한다. 색상만으로 정보를 전달하지 않고 범례, 접근 가능한 제목과 설명, 정확한 표 대체 정보를 함께 제공하며 장식과 애니메이션은 추가하지 않는다.

## 변경하지 않는 범위

금융 공식, 계산 정밀도, 세금 가정, 메타데이터, 사이트맵, canonical URL, 구조화 데이터는 변경하지 않는다. 외부 참고 페이지는 정보 우선순위 연구에만 사용하며 코드, 문구, 시각 스타일을 복제하지 않는다.
